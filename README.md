# RoomieSync

## Project Description
RoomieSync is a shared living space management application designed for student groups and young adults renting shared living spaces. It automates the calculation of bill splitting and chore scheduling, providing a transparent Dashboard to clearly display each individual's responsibilities, helping to preserve friendships and save time.

* Node.js (v16.x or higher recommended)
* npm or Yarn
* Git

## How to Run the Project (Local Environment)

**1. Clone the repository:**
`git clone https://github.com/khoanguyencit23-sketch/RoomieSync.git`

**2. Navigate to the project directory:**
`cd RoomieSync`

**3. Install dependencies:**
`npm install`

**4. Start the application:**
`npm start`

*(The application will be running on http://localhost:3000 by default).*
